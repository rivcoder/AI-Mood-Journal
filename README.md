# AI-Mood-Journal

MoodMind is an AI-powered mood journaling app with a sleek UI.  
Users enter their **mood** and **thoughts**, and the app analyzes them to provide a **positive, uplifting quote**.  
Perfect for tracking emotional wellbeing and spreading positivity.

## Features
- Mood input
- Thoughts input
- Mood analysis
- Positive quote output
- Simple & aesthetic UI
